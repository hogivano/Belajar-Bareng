Petunjuk Instalasi Belajar Bareng.apk (belum tersedia di playstore) update aplikasi intip.in/bebarapk :

Minimum Recruitement :
OS : Jelly Bean (4.2.0)
RAM : 1 GB

1. Kirim file program Belajar Bareng.apk ke smartphone anda 
2. Buka file tersebut, dan akan muncul Petunjuk Instalasi Program
3. Klik tombol next dan akan muncul Installation Progress- nya
4. Setelah proses Instalasi selesai, Program tersebut dapat langsung dijalankan.

Kegunaan aplikasi :
- Untuk mencari kelas belajar bersama dengan user lain
- Jika sudah join user harus mendatangi tempat kelas belajar tersebut untuk belajar bersama
